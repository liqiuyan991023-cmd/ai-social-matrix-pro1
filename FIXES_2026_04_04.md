## 前置条件和已知限制

### 前置条件
1. ✅ Node.js >= 18
2. ✅ npm >= 9
3. ✅ 有效的 API 密钥 (至少 LONGCAT_API_KEY)
4. ✅ Redis 连接 (可选，若使用持久化)

### 已知限制
1. localStorage 在隐私/无痕模式不可用
2. Tavily API 需要有效密钥（无则使用fallback）
3. max-h-[600px] 在小屏幕可能需要调整
4. 多设备无法同步 localStorage 数据

- Chrome/Edge 90+ ✅
- Firefox 88+ ✅
- Safari 14+ ✅
- 需要 ES6+ 支持
